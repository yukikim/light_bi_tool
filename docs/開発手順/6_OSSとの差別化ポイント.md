# 開発手順 6: OSSとの差別化ポイント

OSSのBI/可視化ツールは強力ですが、MVPとしてのLightBIは「使い始めるまでの速さ」と「安全にSQLを実行できる最小設計」を重視します。

## 差別化の軸（MVP向け）

- CSVアップロード → 自動テーブル作成 → クエリ/ダッシュボード/ウィジェットまで自動生成
- SQL実行の安全設計を最初から内蔵（SELECT限定・禁止語・強制LIMIT・timeout・実行ログ）
- 画面導線が最小（ダッシュボード/クエリ/CSVの3導線）

## 将来的に伸ばせる方向

- テナント/権限
- 共有（URL/埋め込み）
- 監査・実行制限
- データソース複数対応
